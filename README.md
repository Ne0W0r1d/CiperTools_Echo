## Cipher Tools - Echo 简介

这是 Cipher Tools 的 Ping 工具，代号 Echo，而中文名叫 **回音之钥**<br>

本项目和 Flathub 现存的 Echo 并没有关系，这是一个基于 Qt 开发的，为 Cipher Tools 设计的 Ping 工具。<br>

本软件属于 Rana Toolkit 的一部分，系 Cipher Tools 工具箱的开发代号。
